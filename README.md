# Cloak Obfuscator
Cloak is a .NET obfuscator written in C#. It's primary goal is to be
more of a learning tool than a good obfuscator, but it can double as both :smile:

## Features
- [ ] String Encryption
- [ ] Control Flow
- [ ] Call Proxying
- [ ] Renaming
- [ ] Int Encryption
